## Product Line: Premium Playbooks

Each playbook solves one specific business problem using AI tools. Playbooks are the core product and the primary revenue driver in Phase 1.

### Pricing

- **Standard playbooks:** $197
- **Deep-dive playbooks:** $297 to $497 (more complex topics, more tools tested, longer implementation)

Pricing reflects the value of the time saved, not the length of the document. A $497 playbook saves the buyer 20+ hours of research, testing, and troubleshooting. That math works for any business owner billing their time at $50+/hour.

### What Makes a Good Playbook Topic

A strong playbook topic has all of the following:

1. **A specific, painful business problem.** Not "how to use AI" but "how to automate client follow-up so leads stop going cold."
2. **Multiple tool options to test.** The playbook should compare at least 2 to 3 approaches or platforms, with a clear recommendation.
3. **A non-technical buyer.** The target reader can follow step-by-step instructions but cannot write code.
4. **Demonstrable time savings.** The playbook should save the buyer a measurable amount of time (10+ hours of setup and testing, ideally 20+).
5. **Ongoing demand.** The topic should be something people are actively searching for, not a one-time trend.

### Playbook Quality Standards

At $197 to $497, the playbook must be genuinely implementation-ready. This is the quality bar:

- **Exact steps.** Every action the reader takes is documented. No "configure as needed" handwaving.
- **Exact tool settings.** Screenshots of every screen. What to click, what to type, what to select.
- **Exact prompts.** If AI prompts are involved, the full prompt text is provided and explained.
- **Troubleshooting guide.** Common failure points and how to fix them. What to do when the tool doesn't behave as expected.
- **Tool comparisons.** Where multiple tools could solve the problem, the playbook tests them and gives a clear recommendation with reasoning.
- **Platform-agnostic where possible.** Playbooks recommend whichever tools actually work best for the problem. No locking to a single platform unless that platform is clearly the best option.

### Format

Playbooks are delivered as digital downloads (PDF or similar). Instant delivery after purchase. No drip, no login, no course platform. Buy it, download it, use it.

## The First Playbook: "How to Build a Done-for-You AI Automation Agency"

All the agency-era work becomes the content of the first premium playbook. This playbook teaches someone how to start and run an AI automation agency using Claude Cowork.

**Price:** $297 to $497 (given the depth of material).

**What it covers:**

- How to define a niche and build an ICP (methodology is transferable; the contractor example is the case study)
- How to structure pricing using the grand slam offer framework (value stacking, founding partner program)
- How to design a two-call close sales process for high-ticket services
- How to build a scope-of-work guarantee instead of a refund guarantee
- How to structure 50/50 payment splits
- How to build Cowork plugins as client deliverables
- How to architect the plugin (skills, commands, MCP connections, README)
- How to handle client onboarding, handoff, and maintenance
- The integration landscape: what tools have MCP connectors, what doesn't, and how to work around gaps

**Source material:** Everything in WM_Legacy/ (BUSINESS.md, SERVICE_PACKAGES.md, PLUGIN_ARCHITECTURE.md, ICP.md, sales templates, the plugin itself). All of it was built as if we were running the agency, which means the playbook reader gets the exact system, not a theoretical framework.
